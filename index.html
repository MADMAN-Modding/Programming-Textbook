What's in here

About this textbook #

This is a self-contained, browser-based textbook designed for use in a high-school Introduction to Programming course. Each chapter is one HTML page, mobile-friendly, print-friendly, and built to be deep-linked from a learning management system.

Every concept is paired with one of Dan Shiffman's Coding Train videos so students can hear an enthusiastic, beginner-friendly explanation alongside the written material. Code samples use a consistent syntax-highlighting palette to make keywords, functions, numbers, and strings visually scannable.

Every chapter ends with a Common Beginner Mistakes section and four practice exercises. The seven chapters build on each other — by Chapter 7, students are combining classes, arrays, and mouse input to build particle systems from scratch.

For teachers

How to use this #

  • Each chapter is one class period of reading + discussion, or a take-home assignment.
  • Pair each reading with live coding in the p5.js Web Editor.
  • Use the practice exercises as in-class lab work or homework.
  • The "Common Beginner Mistakes" sections are great for whole-class debugging discussions.
  • Print-friendly CSS means students can print clean PDFs for offline reading.
